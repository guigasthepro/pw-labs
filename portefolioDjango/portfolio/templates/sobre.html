Sobre este Website

Este website foi criado utilizando as seguintes tecnologias:

  • HTML: Hypertext Markup Language
    Ano de criação: 1991 Criador: Tim Berners-Lee Link para site oficial: HTML Descrição: HTML é a linguagem de marcação utilizada para criar páginas web. É composta por uma série de elementos, como tags e atributos, que definem o conteúdo e a estrutura da página.
  • CSS: Cascade Style Sheet
    Ano de criação: 1996 Criador: Håkon Wium Lie Link para site oficial: CSS Descrição: CSS é uma linguagem utilizada para estilizar páginas web. Permite definir a aparência de elementos HTML, como fontes, cores, bordas, margens e posicionamento.
  • JS: JavaScript
    Ano de criação: 1995 Criador: Brendan Eich Link para site oficial: JavaScript Descrição: JavaScript® (às vezes abreviado para JS) é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe, mais conhecida como a linguagem de script para páginas Web
  • Py: Python
    Ano de criação: 1991 Criador: Guido van Rossum Link para site oficial: Python Descrição: Python é uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte
