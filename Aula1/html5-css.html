HTML 5

O HTML5 é a versão mais recente da Linguagem de Marcação de Hipertexto, que suporta tags, elementos e conteúdo multimídia — assim como marcações

header ✔ Para realizar o topo da página
nav ✔ Para fazer o menu de navegação
main ✔ Para meter o conteudo da página
Article ❌
Section ✔ Para criar secções de texto
aside ❌
footer ✔ Para realizar um "cabeçalho em baixo"
details ❌ Define detalhes dando a possibilidade de esconder
meter ❌ Define uma escala unitária
progress ❌ Define uma barra de progresso
abbr ❌ Representa o significado de uma abreviatura
adress ❌ Informação sobre o autor

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML

universal ✔ Para aplicar em toda a página
tipo ✔ Apenas escolhe o tipo de elemento html
classe ✔ Apenas estiliza o elemento que tenha aquele nome
id ✔ Permite identificar cada elemento do html
atributo ✔ Apenas estiliza os atributos dos elementos
pseudo-classe ❌ Apenas estiliza um estado de um elemento
pseudo-elemento ❌ estiliza o elemento com id que tenha sido referenciado no url , com #
